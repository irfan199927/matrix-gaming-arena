# Workflows

Markdown SOPs that define how to accomplish tasks using the WAT framework.

## Structure of a workflow file

```
# Workflow: <name>

## Objective
What this workflow accomplishes.

## Inputs
- input_1: description
- input_2: description

## Steps
1. Step description → tool: `tools/script.py`
2. ...

## Outputs
What is produced and where it goes.

## Edge Cases / Notes
Known gotchas, rate limits, quirks discovered during use.
```
